# prometheon

NAS terminal AI interface — system info, photo browser, and secure command runner for the PROMETHEUS pool.

## Setup

- Python 3, Flask, see `requirements.txt` and `requirements-ai.txt`
- Copy `.env.example` to `.env` and set `PROMETHEON_PASSWORD`, `ANTHROPIC_API_KEY` (optional)

## Run

```bash
python app.py
```
